Patient Data Access

The Clinical Intelligence API provides programmatic access to anonymized patient records, diagnostic histories, and risk metrics. Use this endpoint to retrieve comprehensive profiles required for population health management and research analysis.

Authentication

All API requests must include your API Key in the Authorization HTTP header. We use standard Bearer token authentication.

Authorization: Bearer DATAVIBE_YOUR_SECRET_KEY
Base URL
PRODUCTION https://api.datavibe.clinic/v2
GET

/api/patients

Retrieve a paginated list of patient records. Filters can be applied to narrow down specific patient cohorts based on clinical markers.

Query Parameters
Parameter Type Description
limit integer Number of records to return. Default is 20. Max 100.
offset integer The starting index for pagination.
risk_score_gt float Filter patients with a risk score greater than the provided value (0.0 to 1.0).
Patient Schema

PatientObject

JSON
id
string(uuid)

Unique identifier for the patient.

clinical_status
enum["active", "monitored", "discharged"]

Current status of the patient in the intelligence pipeline.

last_assessment
iso8601 date

Timestamp of the most recent clinical evaluation.

vitals_summary
object

Condensed object containing average BPM, SpO2, and temp.

Risk Assessment Programmatic Endpoint

Endpoints related to scoring patient cohorts using the Framingham Cardiovascular Risk metrics. Resolves 10-year risk percentages based on biometric measurements.

GET

/api/risk-assessment

Queries the scoring engine database for predictive high-risk cohorts.

Query Parameters
Parameter Type Description
category string Risk categorization filter: "High", "Moderate", "Low".

Dashboard Data Stream

Serves real-time aggregate parameters for public health and clinical analytics panels.

GET

/api/dashboard-data

Returns totals, averages, regional distributions, and active alert objects.

Reports Export Portal

Enables scheduling and downloading generated clinical and policy summaries.

GET

/api/reports

List recent PDF and CSV reports metadata.
